# Scheduler

This project generates and displays a timetable/scheduler.

To run the scheduler, run the main Python file:

```python main.py```

For personalization of the schedule, any changes should be made in the file:

```
py/personal_data.py
```

So the basic workflow is:

1. Open `py/personal_data.py`
2. Make your personalization changes there
3. Run python `main.py`

`main.py` is the entry point of the project, and it handles the process needed to prepare/update the schedule.

Short project overview:

* main.py → the file you run
* py/personal_data.py → personalization settings/data
* py/request.py → fetches raw timetable data
* py/extract_data_from_regulartt.py → extracts timetable data
* py/polish_data.py → cleans/processes the extracted data
* json/ → JSON files used/generated by the project
* index.html / main.js / main.css / js / css → frontend display of the schedule

Requirements:

* Python 3 installed

If an error appears for a missing Python package (for example ModuleNotFoundError), install the missing package and run the program again. Example:

```
pip install requests
```

Notes:

* If you want to refresh/update the schedule data, run python main.py again.
* Personal schedule customization should be done only in py/personal_data.py.
